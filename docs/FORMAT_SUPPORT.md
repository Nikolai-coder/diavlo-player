# Format Support Matrix

Version: 0.1.0 (Phase 1A — format expansion)
Engine: Symphonia 0.6.0 + symphonia-adapter-libopus 0.3.0

Status is ONLY marked PASS after validation with a real fixture file.

## Status Key

- **PASS**: Tested with real fixture, stable playback.
- **PARTIAL**: Decodes but known limitations exist.
- **UNSUPPORTED**: Not supported by current codecs/features.
- **BLOCKED**: Needs external library or license review.

## Limitations

- AAC: encoder delay / gapless not handled (92160 samples vs 88200 expected). Trim is application responsibility.
- Opus: pre-skip handled by the adapter (356 samples at 48 kHz).
- MP3: gapless info not parsed.
- HE-AAC (SBR): not tested. Requires explicit SBR handling — do not claim support.
- Opus in MP4/CAF container: not tested (uncommon).
- Multichannel (>2): not tested for any format.
- High sample rates (>96 kHz): not tested.
