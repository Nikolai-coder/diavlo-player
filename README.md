# DIAVLO PLAYER

**A modern, glass-style music player for Windows.**

---

## ✦ Features

- **Frameless glass window** with DWM blur — dark, sleek, modern
- **Wide format support** — WAV, FLAC, MP3, AAC, OGG, Opus, AIFF, M4A
- **Drag & drop** — drop audio files directly onto the window
- **CLI support** — `diavlo-player.exe song.mp3` works from terminal
- **Set as default player** — registers with Windows for all audio formats
- **Lightweight** — ~12 MB binary, ~60 MB RAM idle

## ✦ Download

Get the latest version from **[Releases](https://github.com/Nikolai-coder/diavlo-player/releases)**.

Download `diavlo-player.exe` and run it — no installer needed (portable).

## ✦ Requirements

```

diavlo-player.exe "song.mp3"         # Open and play a file
```
